# 📝 Redux Toolkit Todo App

A modern Todo application built using React and Redux Toolkit. This project demonstrates scalable state management and clean architecture, along with unique ID generation using nanoid.

---

## 🚀 Features

- Add Todo
- Delete Todo
- Unique ID generation using nanoid
- Global State Management with Redux Toolkit
- Clean and scalable code structure

---

## 🧠 What I Learned

- Redux Toolkit (createSlice, reducers, actions)
- Managing global state in React
- Using nanoid for unique IDs
- Writing maintainable and scalable code

---

## ⚙️ Tech Stack

- React.js
- Redux Toolkit
- JavaScript
- Tailwind CSS
- nanoid
